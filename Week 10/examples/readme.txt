Steps to use the solution files. 
1. set your config.env to your own database connection string and password
2. It is the week10 database that you setup previously.
3. Using terminal, Navigate to each Exercise folder and install all the dependencies as it has not been included to save file space
4. Start up the server using nodemon sever.js
5. go to localhost:8000/Link to check out all the expected output of each exercise.
