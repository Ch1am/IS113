Exercise 10

Combine several CommonJS modules into a small program. This mirrors how larger applications organize code across multiple files.

Tasks

  1. In math.js, export add(a, b) and subtract(a, b).
  2. In constants.js, export constants PI and E.
  3. In formatter.js, export a single function formatResult(x) as the module’s value.
  4. In index.js, import everything, compute add(PI, E), format the result, and log it.

Expected Output

Result: 5.14
