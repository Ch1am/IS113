Exercise 8

Simulate a “default export” style in CommonJS by exporting a single function directly with module.exports = functionName. You can still attach additional functions as properties.

Tasks

  1. In logger.js, define two functions: log(message) and warn(message).
  2. Export log as the main function using module.exports = log.
  3. Also attach warn to the same export object.
  4. In app.js, import and call both functions.

Expected Output

LOG: Hello from Exercise 8
WARNING: Heads up!
