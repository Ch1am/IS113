Exercise 7

Create a module file mathUtils.js that exports multiple functions using module.exports, and import them in app.js with require().

Tasks

  1. In mathUtils.js, define two functions: add(x, y) and multiply(x, y).
  2. Export both using module.exports.
  3. In app.js, import both functions using destructuring.
  4. Log the results of add(2, 3) and multiply(4, 5).
